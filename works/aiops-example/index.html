项目简介

AIOps-example 是一个企业级 AI 工程架构模式展示仓库，场景是模拟的 IT 运维/SRE 场景（故障处置手册、历史工单、告警、变更记录）。仓库把从最基础的 RAG，到进阶混合检索、LangGraph 多智能体（Adaptive RAG、人工审批 HITL、Supervisor 路由）、MCP 工具协议集成、Agent 编排模式对比（ReAct / Plan-and-Execute / Planner-Generator-Evaluator）、企业级 API 网关（RBAC + 限速 + 审计日志 + 多租户）、多智能体框架横向对比（LangGraph vs Smolagents vs CrewAI vs AutoGen），到沙箱化代码执行 Agent，每一种架构模式都做成一个真正跑得起来的独立示例。所有 LLM/Agent 调用接入 Langfuse 追踪，服务指标接入 Prometheus + Grafana 监控，全部数据都是确定性生成的模拟数据，不连接任何真实生产系统。
