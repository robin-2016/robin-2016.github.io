项目简介

write-blog 是一个内容创作与多平台分发的 Claude Code Skill（源自 blogsync 项目）。三种模式生成中文文章——每日新闻精选（展示今日 Hacker News / GitHub Trending / Reddit / X-Twitter 全列表供手动选题）、GitHub 项目解读（自动提取仓库 README 撰写解读文章）、自然语言主题写作（结合 WebSearch 收集资料并做事实核查）。写完后自动生成封面图（HuggingFace FLUX）和知识总结图（DashScope，按内容类型自动选风格），经人工审核后一键发布：微信公众号草稿箱、知乎/掘金/CSDN/头条/微博/百家号等 10+ 平台通过 wechatsync CLI + Playwright 自动提交，以及本博客（GitHub 个人主页）同步更新，覆盖 29+ 分发渠道。触发方式是在 Claude Code 中执行 /write-blog daily-news、/write-blog github <url> 或 /write-blog sentence <描述>，全流程在人工审核步骤暂停等待确认。
