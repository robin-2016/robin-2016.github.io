项目简介

StockMarketAI 是一个美股多周期分层投研系统。自动筛选 S&P 500 + NASDAQ 100 + ETF，根据 VIX、收益率曲线等宏观体制信号动态激活波段（1-4周）、中期（1-3月）、长期（3月+）、ETF 四个持仓池，由 Hermes Agent 定时调度生成 AI 研报并推送。同时通过盈立证券（uSMART）Open API 接入实时行情、持仓、期权链和交易执行，在 Claude Code 中以七个交互 Skill 的形式使用：/portfolio 查询实时持仓与止损预警、/analyze 对指定股票执行多头/空头/宏观/裁判四角色 AI 委员会深度分析、/optstrat 给出期权策略建议、/quote 查看实时行情快照、/kline 文本渲染 K 线图、/trade 半自动交易（买入/卖出/撤单/止损止盈，写操作需用户确认）、/scout 自上而下发现被低估的股票与 ETF。所有输出仅供个人投资研究参考，不构成投资建议。
