Overview

RobinDevNotes is my personal tech learning blog — hands-on notes, hard-learned lessons, and practical summaries from self-studying various technologies. Content covers AI tools in practice (DeepSeek, NotebookLM, Cursor), cloud-native (Kubernetes, CRD, Kubebuilder), infrastructure as code (Terraform, Packer), Go backend (Gin, GORM, PostgreSQL), and Vue 3 frontend. The blog itself is an open-source project, migrated from VuePress to Astro with bilingual Chinese/English support.
