Overview

EnglishRes answers one question: 'I want to improve my English — what tools and resources should I actually use?' The site curates and categorizes resources across six areas: vocabulary, listening, speaking, reading, grammar, and test preparation. Each resource includes a description, recommended use case, and difficulty level. Features real-time keyword search (across names, descriptions, and tags) and one-click language switching between Chinese and English. Built with Astro for static generation and deployed on Cloudflare Pages for fast global access.
