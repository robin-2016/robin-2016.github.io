Go-安装配置

发表于 2018-04-20

1.安装Go
Go的官网：golang.org，但是打不开
可以从Go语言中文网下载，选择对应的平台包进行安装就行。还可以从GO的GitHub下载最新的，不过是源码。

2.安装前要明白的关于环境变量的设置

  • GOROOT
    Go的安装目录
  • GOPATH
    Go的工作区，可以设置多个，这个就是我们编写源码地方
  • GOBIN
    存放Go的可执行文件的目录

3.常用的操作

  • go get
    下载其他go程序，优先下载到第一个GOPATH目录
  • go install
    安装扩展文件，执行文件会放在bin目录中，非执行文件安装到go下的pkg目录
  • go build
    编译源码文件，生成对应平台可执行程序
  • go run
    编译并运行源码文件

4.IDE
我推荐使用JetBrains出品的golang，都是有了这个IDE我才鼓起学习Go的勇气。
